## **📝 How to Use**
1. **Navigate to the problem category** (e.g., `Arrays`).
2. **Open the `.java` file** for the problem you want to review.
3. **Copy the solution** or run it locally.

### **Example: Running a Solution**
```bash
javac Arrays/TwoSum.java
java Arrays.TwoSum
```
